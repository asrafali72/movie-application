# 🎬 MERN Stack Movie Application

A production-ready Movie Management Platform built using the MERN stack with secure authentication and role-based access control.

## 🚀 Key Highlights
- Full-stack MERN application
- JWT Authentication & RBAC
- Admin dashboard for CRUD operations
- Optimized API & pagination
- Responsive UI with Material UI

## 🌐 Live Demo
Frontend: https://movie-application-frontend.onrender.com 

Backend API: https://your-api.onrender.com  

## 🧩 Features

### User
- Browse, search, and sort movies
- Pagination & responsive UI

### Admin
- Secure login
- Add, update, and delete movies
- Protected routes using middleware

## 🛠 Tech Stack
React (Vite) • Node.js • Express • MongoDB Atlas • JWT • Material UI
